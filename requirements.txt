Requirements:
  Python 3.7, pip3, django
  MySQL 5.7.25
  
Tools: PyCharm IDE

Create a db with name get_sql

Run these commands:

1. pip3 install mysql-connector-python


Run - python3 manage.py migrate

Start server by - python3 manage.py runserver

check url - http://127.0.0.1:8000/connection
Response should be  - "Connection Success"


Creating Tables :
1. CREATE TABLE tables (
	id BIGINT(20) NOT NULL AUTO_INCREMENT,
	NAME VARCHAR(225) NOT NULL,
	account_id BIGINT(20) NOT NULL,
	PRIMARY KEY (account_id,id),
	KEY index_table_ids (id),
	KEY index_table_account_and_ids (account_id,id)
);

Description: We will store all the table names in this table and use only its id in other tables;

2. CREATE TABLE `accounts` (
	id BIGINT(20) NOT NULL AUTO_INCREMENT,
	`name` VARCHAR(225) NOT NULL,
	`email` VARCHAR(225) NOT NULL,
	PRIMARY KEY (id),
	KEY index_table_ids (id)
);

Description: Creating a multi-tenant environment which is commonly used accross all cloud apps.

3. CREATE TABLE `columns` (
	account_id BIGINT(20) NOT NULL,
	id BIGINT(20) NOT NULL AUTO_INCREMENT,
	`name` VARCHAR(225) NOT NULL,
	`type` BIGINT(20) NOT NULL,
	table_id BIGINT(20) NOT NULL,
	is_primary TINYINT(1) NOT NULL DEFAULT '0',
	`is_indexed` TINYINT(1) NOT NULL DEFAULT '0',
	foreign_table_id BIGINT(20),
	PRIMARY KEY (account_id,id),
	KEY index_table_ids (id),
	KEY index_table_account_and_ids (account_id,id)
);

Description: To store all the columns of tables.
